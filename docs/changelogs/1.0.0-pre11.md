# Amber - 1.0.0 - Beta 11

- Amber now supports nip 46
- There's 3 versions of the application to download now:
  - Play - This uses the google services push notifications
  - Free - This uses ntfy push notifications
  - Offline - This does not support nip 46
- Support for Orbot proxy
- Add settings to connect to relay directly
- Fix crash when using web applications with nostrsigner: intents
- Fix decryption for web apps using intents
- Added activity history screen
